Per testare l'area admin, partire da: http://localhost:8100/login (rotta di default dell'app).
Per testare l'area customer, partire da: http://localhost:8100/homepage.

Dalla cartella server, digitare il comando npm install per installare l'applicazione.

Client:
Dalla cartella ecommerce, digitare il comando npm install per installare l'applicazione.

Versione di Node usata: v22.17.0
